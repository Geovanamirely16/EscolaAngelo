1 #Como CLONAR um repositório do GITHUB com o Visual Code
2 I No GITHUB abra o repositório e clique no botão CODE> Local copiar link do repositório
3 No Visual Code abra a pasta de seus projetos (exemplo: Documentos/Python/)
4 Abra o Terminal e digite o comando: git clone <cole aqui o link compiado
5 Tecle ENTER
6 No VS Code abra a pasta clonada
7 No terminal digite: git config user.name <digite aqui seu nome de usuário no Github>
8 Tecle ENTER
9 Ainda no terminal digite: git config user.email <digite aqui seu email no GitHub>
10 Tecle ENTER
